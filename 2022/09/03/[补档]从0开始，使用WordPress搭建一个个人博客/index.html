补档|从0开始，使用WordPress搭建一个个人博客

这两天用WordPress搭建了一个个人博客，使用体验十分良好。最开始接触WordPress搭建还是在初中，那会用的是腾讯云的宝塔界面搭建的，总感觉b格不足，后来由于交不起每个月十块钱的服务器租金，博客就挂了。

前期准备

  • 一台Linux服务器：我这里使用的腾讯云的服务器，可以随便选择大厂的服务器，国外服务器速度会慢很多，但貌似绑定域名可以不用备案。
  • 一个远程连接软件：厂商一般会提供网页控制台连接，但如果要经常登陆服务器运维的话，建议下载一个远程连接软件。我用的是Putty。一款免费的软件，使用教程可以百度，有很多。
  • 一个域名：准备一个域名可以告别IP裸连，国内主机需要备案。我嫌麻烦暂时还没有准备这个（）。
  • 学会vim：这个基本上是Linux必会内容，不会的百度自学。

LNMP环境部署

WordPress是一个使用PHP开发的博客平台，这里使用LNMP（Linux + Nginx + MySQL + PHP）部署环境。

  • 安装LNMP脚本：目前最新版本是1.9版本

    wget http://soft.vpser.net/lnmp/lnmp1.9.tar.gz -cO lnmp1.9.tar.gz

  • 解压执行脚本：

    tar zxf lnmp1.5.tar.gz && cd lnmp1.5 && ./install.sh lnmp

运行脚本后会出现许多如下的选项，如果没有特殊需求，按回车选择默认即可

设置Mysql数据库root用户密码时不要直接跳过（捂脸），这里输入你要设置的数据库密码

选择PHP版本的时候推荐选择最高版本的PHP，这样之后不用很麻烦地去升级PHP，默认PHP版本现在WordPress大多数主题已经不支持了

之后便是一路回车，等待安装，安装时间大概在半个小时左右，随网速等原因变化

最后出现如下界面便代表安装成功（✌）

WordPress环境配置

  • 安装WordPress并解压到`/home/wwwroot``

    wget https://cn.wordpress.org/latest-zh_CN.zip && unzip latest-zh_CN.zip -d /home/wwwroot

  • 登陆Mysql数据库，密码就是之前设置的那个，创建一个WordPress的表

  • 修改nginx文件位置（使用vim）

    vim /usr/local/nginx/conf/nginx.conf

​ 找到图下红框位置，将/home/wwwroot/default改为`/home/wwwroot/wordpress

​ 输入`nginx -t` 验证nginx配置文件是否有误，

​ 如果出现successful和ok字样后无误后输入nginx -s reload 重新加载nginx

  • 更改WordPress文件权限

    cd /home/wwwroot && chown -R www wordpress/ && chgrp -R www wordpress/

  • 配置WordPress

​ 到这里配置基本完成，在浏览器输入http://ip/wp-admin/setup-config.php（这里ip为你的服务器 ip）

​ 这里数据库名填wordpress，主机填127.0.0.1，密码为数据库root密码

​ 这里用户名和邮箱一定记住，后续登陆要用

访问网站

在浏览器输入http://ip即可登陆博客主页,输入http://ip/wp-admin即可登陆后台管理界面

写在后面

这样一个属于自己的个人博客就算搭建完成了，但是wordpress的原生外观可能不会得到太多人的喜欢。可以在后台管理系统中的外观更换主题，也可以安装插件使得博客功能更多。（后续可能会出一篇讲讲如何升级php版本）
